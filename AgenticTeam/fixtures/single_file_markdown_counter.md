# Project: Single File Markdown Counter

## Goal
Build a tiny CLI tool that counts lines, words, and characters from one markdown file.

## Tech Stack
- Python 3.x
- Standard library only

## Requirements
1. Accept one file path from the command line.
2. Print line, word, and character counts.
3. Keep the implementation minimal and deterministic.

## Acceptance Criteria
1. `README.md`, `src/main.py`, and `tests/test_main.py` are present.
2. Running the tool on a markdown file prints the three counts.
3. Tests cover the counting behavior.

## Determinism Rules
- Keep the output format stable.
- Do not create additional tasks unless blocked.
