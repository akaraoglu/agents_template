# Project: Minimal Python CLI

## Goal
Build a very small Python CLI using only the standard library.

## Requirements
1. Provide a single entrypoint script under `src/`.
2. Support one deterministic command-line action.
3. Include a short README and one test module.

## Acceptance Criteria
1. The CLI runs with Python 3.
2. The repository contains `README.md`, `src/main.py`, and `tests/test_main.py`.
3. Tests can be executed without external services.

## Determinism Rules
- Keep the project small.
- Do not invent extra phases or files unless blocked.
