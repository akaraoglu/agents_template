# Stories

Create one file per story in this folder.

Recommended naming:
- `STORY-001-short-title.md`
- `STORY-002-short-title.md`

Each story should:
- link to a milestone or explicit goal
- define user or system value
- include acceptance criteria
- link to its tasks
- move through clear statuses such as `draft`, `ready`, `in_progress`,
  `done`, and `verified`
